About

EveryChat is a useful application that allows a number of people on the World Wide Web to talk to one another simultaneously. The ability to chat on the Web can be a quick way to hold a virtual meeting. EveryChat was designed to be as simple as possible for everyone and it is one of the most lightweight chat scripts out there. It takes up no more resources than a standard form submit script and it will run on any server that supports Perl. EveryChat has been tested on UNIX, Linux, Windows, BeOS and Macintosh platforms under many different web servers. On the user end, EveryChat will run under any browser.
